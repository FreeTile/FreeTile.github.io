About Me

Have you ever heard about the best game developer in the world? That’s not me, but I’ll try my best to become him. one day. My name is Mark, I was born in Russia and moved to Canada to make my dream come true and create new video games.

I really love inventing new worlds and concepts and implementing already existing mechanics into them. At some moments this is a kind of recreation for me.

Right now I am developing a slasher game. Although it is not published yet, I’m still proud I undertook this complex project, it fully shows my determination to create games and my skills as a game developer. This project fully shows my skill set with Unity 3D, Blender (modeling and animations) and C#. It also reflects my ability to solve problems. I also have a good communication level, though I don’t have a fairly high level of language.

I really believe that games are not only about money. It’s like art, but in the digital world, and I hope, in the future I will create not regular games, but really unique creations that can bring new experiences for everyone. I would be glad to join a team to create games and learn something new. This would allow me to fulfill my potential in the game industry.

My Works

AI Cryptotrading Bot

Using Python libraries such as TensorFlow, I developed and trained a model for automated trading on Binance using API.

To train the model, I rented servers with an A100 card from Nvidia. These maps were specifically designed for training neural models. Training models turned out to be quite expensive, so at this stage this project has been put on the back burner for future ideas.

On Github you can find the basic version of the program with genetic selection files, training of the best individual and a script for trading on Binance using the Binance API.

Robot Rampage

Robot rampage is a 3D first person shooter where you have to survive infinite waves of robots on the small arena. You can use weapons such as assault rifle to kill them. There are also some pickups which can give you ammo, health and armor.

This game was created step by step from guide book for Unity 3D.

Bobble Head Wars

This is a 3d person top-down shooter. You have to kill all aliens to pass level and go to the next one. You can grab power up, which spawns somewhere on the battle field.

This game also was created from the book, which I mentioned in the previous project.
